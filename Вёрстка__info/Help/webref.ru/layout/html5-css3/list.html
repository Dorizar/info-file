Списки

Главная / Вёрстка / HTML5 и CSS3 на примерах / Списки

Списком называется взаимосвязанный набор отдельных фраз или предложений, которые начинаются с маркера или цифры. Списки предоставляют возможность упорядочить и систематизировать разные данные и представить их в наглядном и удобном для пользователя виде.

Создание списка

Любой список представляет собой элемент <ul> (от англ. unordered list — неупорядоченный список), который устанавливает маркированный список, или <ol> (от англ. ordered list — упорядоченный список), который определяет список нумерованный. Каждый элемент списка должен начинаться с элемента <li> (пример 1).

Пример 1. Создание списка

<!DOCTYPE html>
<html>
 <head>
  <meta charset="utf-8">
  <title>Списки</title>
 </head>
 <body>
  <p>Текст до списка.</p>
  <ul>
   <li>Сепульки</li>
   <li>Сепулькарии</li>
   <li>Сепуление</li>
  </ul>
  <p>Текст после списка.</p>
 </body>
</html>
