padding в CSS

Главная / Учебные курсы / Блочная модель в CSS / padding в CSS
  • Описание
  • Задания

Поле (padding) — это пространство между рамкой элемента и его содержимым.

Объём пространства может быть определён с помощью любых единиц размера.

blockquote { padding: 20px; }

Как и для границ, padding может быть установлен индивидуально для любой из четырёх сторон.

blockquote { padding-bottom: 20px; }

Поскольку поле лежит между границей и содержимым, внутреннее пространство проще визуализировать применением границ:

blockquote { background: yellow; border: 1px solid blue; }

Добавление полей обеспечивает пространство между текстовым содержанием и границами:

blockquote { background: yellow; border: 1px solid blue; padding: 20px; }

Обратите внимание на то, что фон элемента растягивается до его границ. Применение полей позволяет расширить этот фон.
