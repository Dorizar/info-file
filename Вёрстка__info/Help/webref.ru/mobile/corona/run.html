Запуск программы

Главная / Мобильные приложения / Делаем приложение на Corona SDK / Запуск программы

При запуске Corona открывается два окна. Первое это консоль вывода, в ней отображается техническая информация, а также показываются строки из функции print. Русский язык не поддерживается и отображается кракозябрами. Во втором окне показываются список ваших последних проектов, ссылки на документацию и примеры, номер версии программы, а также кнопки для создания и открытия проекта.

Окно программы

Если вы запустили Corona в первый раз, то рекомендуем начать обзор со ссылки «Samples», где в примерах показываются основные возможности графического движка. При этом открывается такое окно, где можно выбрать желаемый пример.

Окно с примерами

Сами примеры установлены в ту же папку что и сама программа, поэтому понравившийся пример можно скопировать к себе взяв его за образец. Для возвращения к основному окну выберите в меню пункт View > Welcome Screen.

Пример в симуляторе смартфона

В том же меню можно выбрать устройство вывода и посмотреть, как будет выглядеть приложение на разных устройствах.

Выбор устройства вывода

Для увеличения масштаба используйте меню View > Zoom In, а для уменьшения — View > Zoom Out. Масштаб также настраивается клавишами Ctrl++ и Ctrl+-.

Для открытия своего проекта нажмите кнопку Open на экране приветствия или используйте меню File > Open Project... Новый проект создаётся аналогично, только при этом появляется следующее окно.

Окно создания нового проекта

Здесь вы указываете название вашего приложения и папку проекта в которой будут сохраняться все необходимые файлы. Кроме того можно изменить ориентацию экрана по умолчанию — Upright (портретная) и Sideways (ландшафтная), задать размер экрана в пикселах и выбрать готовый шаблон. Ориентацию экрана и его размер можно будет изменить позже, так что оставляйте эти вещи по умолчанию, если не знаете точно что ставить.

Шаблоны предлагаются следующие.

Blank
Пустой шаблон, содержащий только минимум необходимых файлов без какого-либо содержимого. Этот шаблон рекомендуется пользователям уже имеющим опыт работы с Corona SDK.

Tab Bar Application
Шаблон для приложения содержащего вкладки с возможностью переключения между ними.

Physics Based Game
Шаблон для игры со встроенным физическим движком Box2d.

eBook
Шаблон для разработки интерактивных электронных книг.

Multiscene Application
Шаблон демонстрирует работу с разными сценами и переключением между ними.
