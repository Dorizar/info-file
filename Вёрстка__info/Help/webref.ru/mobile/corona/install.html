Установка Corona SDK

Главная / Мобильные приложения / Делаем приложение на Corona SDK / Установка Corona SDK

Скачать последнюю версию программы можно с сайта Corona Labs по адресу coronalabs.com. Предварительно требуется зарегистрироваться, без этого программу не скачать и не запустить. Достаточно указать адрес электронной почты и желаемый пароль.

Форма регистрации и входа на сайте Corona Labs

Скачать программу под Windows или Mac можно по прямой ссылке.

https://developer.coronalabs.com/downloads/corona-sdk

Для работы программы под Windows требуется установленная 32-битная версия Java 6 Development Kit. Заметьте, нужна именно 32-битная версия и именно шестая! Иначе Corona SDK просто не установится и не запустится. Скачать JDK можно по этой ссылке.

http://www.oracle.com/technetwork/java/javase/downloads/java-archive-downloads-javase6-419409.html

При первом запуске программа потребует Email и пароль, который необходимо ввести в форме.

Форма входа

В дальнейшем логиниться в программу уже не нужно, потому что она привязывается к текущему компьютеру. Есть в этом и обратная сторона. Если вы переустановили систему на компьютере, то установить заново Corona SDK не получится. Перед переходом на другой компьютер или свежеустановленную систему нужно заранее провести так называемую деавторизацию и тогда можно уже ставить программу по новой. Если деавторизацию предварительно не сделать, то программу запустить не удастся и придётся обращаться в службу поддержки.

Сама установка программы не представляет никаких сложностей. Принимаем условия лицензионного соглашения и указываем папку установки.
