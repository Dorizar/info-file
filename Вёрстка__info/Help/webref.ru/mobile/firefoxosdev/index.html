Разработка для Firefox OS

Главная / Мобильные приложения / Разработка для Firefox OS

Узнайте, насколько легко и быстро происходит разработка приложений для Firefox OS, новой мобильной операционной системы от Mozilla. Придерживаясь практического подхода этой книги вы научитесь через примеры, как пройти весь путь разработки приложения с самого начала до его дистрибуции в Firefox Marketplace.

Для кого рекомендуется

Всем, кто заинтересован в разработке приложений для новой свободной мобильной платформы. Тем, кто хочет узнать больше о создании приложений с использованием HTML5. Каждому, что желает воспользоваться возможностью, чтобы проложить собственный путь на новом мобильном рынке.

Требуемые знания

Чтобы лучше насладиться этой книгой читатель должен иметь средние знания об HTML, CSS и JavaScript.

Иллюстрация обложки

Обложку создал Рафаэль Экхардт, дизайнер и иллюстратор из Бразилии. Вы можете связаться с ним (он фрилансер) и посмотреть его работы на http://raphaeleckhardt.com.
