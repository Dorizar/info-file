Отменить выделение элементов. [на css3]

Свойства


-webkit-user-select

Значения


auto : Пользователь может выделить элемент.

none : Пользователю запрещено выделять элемент.

text : Пользователь может выделить текст в элементе.

Пример


body { -webkit-user-select: none; }

.enable { -webkit-user-select: text; }




Подсветка активного пункта меню [на jQuery]

Описания примера


Когда создаём любое меню. и делаем для нево все необходимое и стили.
и начинаем мисать скрипт на jq


Сначало указываем путь какому элементу будит применятца этат скрипт
к примеру $(".button a")
дальше приписоваем Обработчик событии
к примеру .click();
дальше вписуем вскопках функцию
к примеру (function(){});
и пишим ищёрас путь к нашему меню
к примеру $(".button a")
патом даписуем клас каторый будеть удалятца сменю
к примеру .removeClass("active");
дальше пишем снова путь к нашей меню Внимания пишем это так
$(this)
ативерь нам нужно добавить класс каторый будет я влятс активным при крике на меню
.addClass("active");

Пример


$(".menu a").click(function(){

$(".menu a").removeClass("active");

$(this).addClass("active");

}); }

Якорные ссылки с анимации Скролла прокрутки [на jQuery]

Описания примера


Якорные ссылки с анимации

Пример


$(document).ready(function(){

$('a[href^="#"]').on('click',function (e) {

e.preventDefault();



var target = this.hash,

$target = $(target);



$('html, body').stop().animate({

'scrollTop': $target.offset().top }, 900, 'swing', function () {

window.location.hash = target;

});

});

});
